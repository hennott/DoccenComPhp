Doccen Funktionen

Die Programmdateien von Doccen kann man leicht weiterverwenden, indem man diese in eigene Projekte einbindet. Dazu kann man diese per require_once verwenden und entsprechend der Dokumentation aufrufen. Doccen selbst ist modular aufgebaut und funktionsorientiert strukturiert. Als Voraussetzung gilt neben den typischen Webtechniken HTML, CSS und JavaScript ebenso PHP.
