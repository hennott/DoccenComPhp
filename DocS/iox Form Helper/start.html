AutoDoc für iox FormHelper

Warum mit AutoDoc dokumentieren

Mit Hilfe von AutoDoc wird eine API-nahe Dokumentation direkt aus dem Quellcode erzeugt. Vordergründig wird dies gemacht, um überhaupt eine Dokumentation zu haben, da diese sonst meist sehr aufwändig ist. Da man den Sourcecode ohnehin dokumentieren muss, wird durch eine spezielle Notation in diesem Aufwand gleich eine besser lesbare Dokumentation erzeugt.

AutoDoc wurde eigens für die Schulungsvideosserien von Share2Brain.com entwickelt und mehr Infos finden sie unter autodoc.share2brain.com
